# Personal Portfolio Website

This project is my submission for **Task 1 of the Full Stack Web Development Internship at Future Interns**.

The goal of this task was to build a professional portfolio website that showcases my skills, projects, and contact information. This portfolio serves as my digital resume where recruiters, clients, and collaborators can explore my work.

## Features

- Responsive design for mobile and desktop
- Home section introducing who I am
- About section with my background
- Skills section listing technologies I use
- Projects section showcasing my work
- Contact section with email and links
